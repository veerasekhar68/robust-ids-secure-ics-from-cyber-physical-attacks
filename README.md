#  Robust IDS for Securing Industrial Control Systems Against Cyber-Physical Attacks

##  Project Overview

This project focuses on protecting **Industrial Control Systems (ICS)** from cyber-physical attacks using an Intrusion Detection System (IDS).

Industrial environments such as SCADA systems, PLC-based automation, and critical infrastructure are vulnerable to cyber threats that can cause real-world physical damage. This system detects malicious activities using anomaly detection techniques and provides a web-based interface for monitoring and analysis.

---

## 🏭 Problem Statement

Cyber-physical attacks on ICS can lead to:

- Equipment damage
- Production downtime
- Financial loss
- National security risks

Traditional IT security solutions are not sufficient for ICS environments. This project proposes a robust IDS system specifically designed for ICS security.

##  Installation & Setup

1. Clone the repository:

```
git clone https://github.com/veerasekhar68/Robust-IDS-Secure-ICS-From-Cyber-Physical-Attacks.git
```

2. Navigate to project folder:

```
cd Robust-IDS-Secure-ICS-From-Cyber-Physical-Attacks
```

3. Install dependencies:

```
pip install -r requirements.txt
```

4. Run server:

```
python manage.py runserver
```

5. Open browser:

```
http://127.0.0.1:8000/
```
